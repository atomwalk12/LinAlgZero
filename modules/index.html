    __init__(config) ¶

    Initialize the base trainer with configuration and setup core components.

    Sets up device selection, session management, logging, and initializes all trainer components to None. Subclasses must implement abstract methods to create model, optimizer, dataloaders, metrics, and loss function.

    Parameters:

    Name Type Description Default
    config ZeroConfig

    Configuration object containing training parameters, model settings, and experiment configuration.

    		 required

    train() ¶

    Execute the complete training loop with validation and checkpointing.

    Runs setup(), then iterates through training steps for config.train_iterations. Handles periodic logging, validation, and checkpoint saving based on config intervals. Automatically manages model train/eval modes and W&B logging.

    visualize_batch(model_output, batch, mode) abstractmethod ¶

    Log visualizations of model predictions and data to W&B.
